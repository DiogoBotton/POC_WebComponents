# POC_WebComponents
Repositório com a finalidade de estudar e realizar POC's envolvendo Web Components

### Criar app React com Vite

```bash
# Com typescript
pnpm create vite@latest my-app -- --template react-ts

# Sem typescript
pnpm create vite@latest my-app -- --template react
```
